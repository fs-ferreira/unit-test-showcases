Bem-vindo ao meu showcase de Testes Unitários

Instruções para Executar o Projeto

  1. Clone o Repositório:
    2. git clone https://github.com/fs-ferreira/unit-test-showcases.git
cd unit-test-showcases
  2. Instale as Dependências:
    3. npm install
  3. Para executar os Testes:
    4. ng test
  4. Caso queira visualizar os Relatórios de Cobertura de Testes:

    5. Após a execução dos testes, você pode visualizar os relatórios de cobertura para garantir que todos os casos de teste sejam cobertos.

    ng test --code-coverage

    Os relatórios estarão disponíveis em unit-test-showcases/coverage/index.html.

  5. Explorando o Código de Teste:

    6. Sinta-se à vontade para explorar os arquivos de teste em unit-test-showcases/src/app. Lá você encontrará todos os testes unitários separados por funcionalidade.

  6. Contribua e Relate Problemas:

    7. Qualquer forma de feedback e ideias para icrementar no projeto são bem vindas!

Acesse o source em https://github.com/fs-ferreira/unit-test-showcases
